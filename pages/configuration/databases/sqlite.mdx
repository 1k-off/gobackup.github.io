# SQLite

GoBackup uses `sqlite` utility to export SQLite database into a `.sql` file.

## Install tool




<Tabs items={['Ubuntu Server', 'Alpine Linux', 'macOS']}>
  <Tab>
    So you need to install `sqlite3` first.

    ```bash
    $ sudo apt install sqlite3
    ```
  </Tab>
  <Tab>
    ```bash 
    $ sudo apk add sqlite
    ```

    https://pkgs.alpinelinux.org/packages?name=sqlite
  </Tab>
  <Tab>
    Install via Homebrew:

    ```bash
    $ brew install sqlite
    ```

    https://formulae.brew.sh/formula/sqlite
  </Tab>
</Tabs>

## Config keys

`type: sqlite`

- `path` - The path of the SQLite 3 database file, for example: `/data/db.sqlite3`.

https://github.com/gobackup/gobackup/blob/main/database/sqlite.go

## Configuration Example

```yml
models:
  my_app:
    databases:
      my_app:
        type: sqlite
        path: /var/data/my_app.sqlite3
```
